# LZJCO-SOEN341_Project_W26
MealMajor : A Meal Prep and Healthy Living Planner

https://lzjco-soen-341-project-w26.vercel.app

MealMajor is a web app for students to plan meals, track groceries, and propose easy recipes.

For the first Sprint : 6 february

Baseline features:
1. User Account Management
- User registration and login
- Profile management (diet preferences, allergies)
